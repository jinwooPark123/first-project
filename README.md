## 프로젝트 배경 정리

### 문제 인식

- 최근 AI 의존성이 과도하게 높아짐
- 공공 과제나 리포트 등이 무분별하게 생성형 AI에 의해 작성됨
- 결과적으로 학생 개개인의 사고력·표현력·문장력 약화 문제가 발생함

---

### 문제 해결 방향

- 단순히 AI가 대신 써주는 도구가 아니라, 창의성을 유지하면서 글쓰기를 보조하는 도구를 만들 필요가 있음
- 학생이 AI의 결과를 수용자가 아닌 사고 주체로서 활용하도록 설계
- 글의 논리와 문맥을 함께 점검하면서 자기 글쓰기 역량을 발전시키는 시스템 구상

---

### 목표 의도

- AI를 ‘대체 수단’이 아니라 ‘학습적 촉진제(assistant for thinking)’로 재정의
- 학생이 AI의 제안을 활용해 자주적이고 비판적인 글쓰기 연습을 하게 함
- 이를 통해 실용적 학습 보조도구이자 창의적 사고 유지 장치로 기능하는 프로그램을 완성

---

### 한줄 요약

> “AI가 글을 대신 쓰는 시대에,  
> 스스로 사고하고 다듬는 힘을 길러주는 ‘창의형 글쓰기 보조 AI’가 필요하다.”
